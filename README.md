# Circuitos

Este repositorio contiene una colección de proyectos de circuitos electrónicos.

## Proyectos

- **Compuertas Logicas:** Contiene proyectos relacionados con compuertas lógicas.
- **Led con LDR que Detecta Luz:** Un circuito con un LED que se enciende o apaga dependiendo de la luz detectada por un LDR.
- **Modulo del Rele:** Un circuito que utiliza un módulo de relé.
- **Prender Led con Transistor:** Un circuito para encender un LED utilizando un transistor.
- **Semaforo con Pulsador (Arduino):** Un semáforo controlado por un pulsador utilizando Arduino.

Cada carpeta de proyecto contiene:
- `Componentes.csv`: Una lista de los componentes necesarios para el circuito.
- `Esquema.pdf`: El esquema del circuito.
- `Codigo.ino` (en algunos casos): El código para el microcontrolador.
